# Musicly

Musicly est un agrégateur de service musicaux.
Grâce à lui vous pouvez consulter vos playlists, recherchez des titres, albums.

## Fonctions prévues:

### Recherche
* Recherche de titres
* Recherche d'artistes
* Recherches d'albums

### Affichage 
* Voir les musiques dans une playlist

### Modification
* Ajouter une musique dans une playlist
* Retirer une musique dans une playlist
